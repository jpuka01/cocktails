Cocktail Recipes

API Description

API Name: TheCocktailDB

Website: This API provides a list of cocktail recipes. Each recipe includes the name of the cocktail, an image of the cocktail, the ingredients, and the instructions to make the cocktail.

Website: https://www.thecocktaildb.com/api.php

Parameters Used: This page uses the s parameter in a GET request to search for cocktails by name. For example, setting s=margarita retrieves all cocktails containing "margarita" in their names.

Potential Applications:

  • Creating a cocktail recipe app or website where users can search for drinks by name, ingredients, or category.
  • Developing a bartending guide that helps users find recipes and instructions to make cocktails at home.

What were the most challenging and most satisfying things about doing this assignment?

The most challenging part of this assignment was understanding how to use the API and parse the JSON data to display the cocktail recipes on the webpage. It required reading the API documentation and experimenting with different parameters to get the desired results.
